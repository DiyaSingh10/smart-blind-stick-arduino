# Smart Blind Stick for Visually Impaired

## About
An assistive navigation system using ultrasonic sensor + Arduino Uno 
to detect obstacles and alert via buzzer.

## Tools Used
Arduino, Embedded C, Ultrasonic Sensor, Wokwi, Tinkercad

## How it Works
- Measures distance using pulse-based signal processing
- Triggers buzzer when object is within threshold distance
